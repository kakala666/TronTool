# TRON USDT 资金池付款系统

## 项目概述

创建一个基于TRON区块链的智能合约付款系统，允许公司集中管理USDT资金，员工通过授权可以使用网页界面从资金池向客户转账。所有转账的链上付款方显示为合约地址，实现资金集中管理。

## 核心需求

### 1. 智能合约功能
- [x] 部署资金池合约到TRON链
- [x] 支持接收USDT充值
- [x] 支持授权地址从资金池转账
- [x] 链上显示付款方为合约地址
- [x] 白名单管理（添加/移除员工地址）
- [x] 余额查询功能

### 2. 网页操作界面
- [ ] 创建员工操作页面
  - [ ] 连接钱包功能（TronLink/imToken）
  - [ ] 输入收款地址
  - [ ] 输入转账金额（USDT）
  - [ ] 执行转账按钮
- [ ] 创建管理员页面
  - [ ] 查看合约余额
  - [ ] 添加/移除员工授权
  - [ ] 查看转账记录
- [ ] 全中文界面
- [ ] 移动端兼容（imToken内置浏览器）

### 3. 部署文档
- [ ] 主网部署指南
- [ ] 员工操作手册
- [ ] 管理员操作手册

## 技术栈

- **区块链**: TRON
- **智能合约**: Solidity ^0.8.6
- **前端**: HTML + JavaScript + TronWeb
- **钱包**: TronLink / imToken

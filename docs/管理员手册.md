# 管理员手册

## 一、职责概述

管理员（Owner）负责：
- 合约充值管理
- 员工授权管理
- 系统运维监控

---

## 二、管理后台使用

### 2.1 访问后台

1. 使用**Owner钱包**（部署合约的钱包）
2. 打开 `admin.html` 页面
3. 自动验证Owner身份
4. 显示管理界面

### 2.2 查看资金池余额

- 顶部卡片显示当前USDT余额
- 余额不足时需要及时充值

### 2.3 添加员工授权

1. 在"员工授权管理"区域
2. 输入员工的TRON地址
3. 点击"添加授权"
4. 钱包确认交易
5. 添加成功后显示在列表中

### 2.4 移除员工授权

1. 在授权列表中找到员工
2. 点击"移除"按钮
3. 确认弹窗中点击"确认"
4. 钱包确认交易

### 2.5 检查地址授权

1. 在"检查地址授权状态"区域
2. 输入要检查的地址
3. 点击"检查"
4. 显示授权状态

---

## 三、资金管理

### 3.1 充值资金池

1. 打开TronLink或imToken
2. 选择USDT (TRC20)
3. 转账
4. 收款地址填：**合约地址**
5. 输入金额
6. 确认发送

### 3.2 余额监控

建议设置余额预警：
- 余额低于1000 USDT时及时充值
- 定期检查资金池状态

### 3.3 查看转账记录

1. 打开 https://tronscan.org
2. 搜索合约地址
3. 点击 **Transfers** 标签
4. 查看所有转账记录

---

## 四、安全管理

### 4.1 Owner钱包安全

**绝对重要！**

- 🔒 使用硬件钱包存储（推荐）
- 🔒 多重备份助记词
- 🔒 不在公共设备上操作
- 🔒 定期更换设备密码

### 4.2 员工授权管理

- 📋 维护员工地址对照表
- 📋 员工离职时及时移除授权
- 📋 定期审查授权列表

### 4.3 资金安全

- 💰 不在合约中存放过多资金
- 💰 分批充值，按需补充
- 💰 定期核对链上记录

---

## 五、运维操作

### 5.1 日常巡检

每日检查：
- [ ] 资金池余额
- [ ] 最近转账是否正常
- [ ] 员工反馈的问题

### 5.2 问题处理

**员工无法转账：**
1. 检查员工是否在白名单
2. 检查资金池余额
3. 检查员工TRX余额

**网页无法访问：**
1. 检查服务器状态
2. 检查HTTPS证书
3. 检查域名解析

### 5.3 紧急情况

如果发现异常转账：
1. 立即检查授权列表
2. 移除可疑地址授权
3. 保存链上证据
4. 联系相关人员

## 六、技术信息

### 合约方法

| 方法 | 作用 | 权限 |
|-----|------|------|
| `payout(to, amount)` | 转账 | 授权员工 |
| `addStaff(address)` | 添加员工 | Owner |
| `removeStaff(address)` | 移除员工 | Owner |
| `getBalance()` | 查询余额 | 所有人 |
| `isAuthorized(address)` | 检查授权 | 所有人 |
